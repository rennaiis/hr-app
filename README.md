# HR test task
Приложение, позволяющее редактировать/смотреть/удалять/добавлять сотрудников в таблицу, редактировать списки профессий и отделов. 

Технологии:`node.js` (`express`, `sqlite3`) для бэкенда. `Vue.js`, `typescript` и `css` для фронтенда 

## Чтобы запустить приложение на компьютере: 
1. Клонируйте репризиторий
2. В vs.code установите нужные зависимости (и в папке frontend, и в папке backend введите `npm install`)
3.  Введите в командной строке `npm start`
